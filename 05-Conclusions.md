## Conclusiones

### Conclusiones y recomendaciones

**TB1:**
El desarrollo de la documentación para crear una aplicación móvil que permita solucionar una problemática real, siguiendo con las indicaciones e instrucciones del enunciado del trabajo, permitió aplicar la teoría de varios marcos de trabajo para lograr el avance que se presenta actualmente. Mediante la investigación de antecendentes y problemática, se pudo investigar los objetivos que tiene nuestra empresa SANUVI con el marco Lean UX y su representación en canvas. La creación de User Persona en base a entrevistas permitió conseguir la información necesaria para realizar el Event Storming, User Stories y el Impact Mapping. Finalmente, desarrollamos la arquitectura de software para nuestra aplicación móvil y los Bounding Context que nos permitirá desarrollar con facilidad nuestra app.
